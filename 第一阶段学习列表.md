# Redis

redis 在 centOs 环境下 启动 关闭命令

```shell
# 启动服务
src/redis-server redis.conf
# 验证启动是否成功
ps -ef | grep redis
# 进入 redis 客户端
src/redis-cli
# 退出客户端
quit
# 推出 redis 服务
pkill redis-server
kill 进程号
src/redis-cli shutdown
```
